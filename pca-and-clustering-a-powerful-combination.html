PCA and clustering a powerful combination!

Permalink: 2015-09-12 08:00:00+01:00 by Michy in Posts tags: R MachineLearning

Example of post

Applying PCA and clustering on the IRIS dataset.

data <- scale(iris[,1:4])
rownames(data) <- iris[,5]
pc <- princomp(data)
plot(pc)
plot(pc,type='l')
summary(pc)

Two components will suffice for our analysis since they explain most of the variance

Let's run PCA

# Get principal component vectors using prcomp instead of princomp
pc <- prcomp(data)
comp <- data.frame(pc$x[,1:3])
plot(comp,col=rgb(0,0,0.5))

As you can see below, most of the variance is captured by PCA:

library(RColorBrewer)
library(scales)

# Apply k-means with k = 3
k <- kmeans(comp, 3, nstart=25, iter.max=1000)
palette(alpha(brewer.pal(9,'Set1'), 0.5))
plot(comp, col=k$clust, pch=4)

# Cluster sizes
sort(table(k$clust))
clust <- names(sort(table(k$clust)))

# First cluster
row.names(data[k$clust==clust[1],])
data[k$clust==clust[1],]
# Second Cluster
row.names(data[k$clust==clust[2],])
data[k$clust==clust[2],]
# Third Cluster
row.names(data[k$clust==clust[3],])
data[k$clust==clust[3],]

As you can see, the clustering algorithm has found 3 clusters and by printing the clusters we see that the algorithm has done a great job of separating the data.
